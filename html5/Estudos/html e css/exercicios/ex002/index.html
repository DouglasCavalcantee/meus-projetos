Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags p e /p

Se precisar quebrar o texto em algum lugar específico como esse, você pode esar a tag br. Ela é uma tag de quebra de linha, ou seja, ela pula para a próxima linha.
E continua escrevendo normalmente.

Você pode usar quantas tags br quiser para criar mais quebras de linha.
